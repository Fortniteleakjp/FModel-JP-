# FModelJP リリースノート

FModelJP をご利用いただきありがとうございます！
このページは、アップデート後に一度だけ表示されます。

## {version} 主な変更点

- FModelJP 5.0 としてメインUIを全面刷新し、全体のレイアウトと操作感を見直しました
- ダークテーマ / ライトテーマの切り替えに対応し、設定画面や各種サブウィンドウのデザインも新UI基準で統一しました
- ロビー背景APIウィンドウを追加し、ロビー背景の確認に加えてフォートナイトのサーバー監視もできるようになりました
- **CloudStorage Hotfix** の適用機能を追加し、開いているアセットへホットフィックスを手動・自動で反映できるようになりました
- 検索画面を強化し、除外ワード指定・アセット種別フィルター・正規表現検索の改善で目的のファイルを探しやすくしました
- エクスポートカートと右クリックメニューを追加し、複数アセットの一括管理とエクスポートがしやすくなりました
- アニメーション周りの読み込み処理を改善し、一部アニメーションや関連アセットがより正常に開けるようになりました
- **アイコンレイアウト編集** を追加し、生成アイコンの名前・説明・画像・背景を自由に配置・装飾できるようになりました

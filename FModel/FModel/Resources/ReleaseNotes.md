# FModelJP リリースノート

FModelJP をご利用いただきありがとうございます！
このページは、アップデート後に一度だけ表示されます。

## {version} 主な変更点

- CUE4Parse コア同期
- (フォートナイトの最新バージョンにCUE4Parseを対応)
- 3D Viewer の改善
- 3D Viewer の UIを全面的に見直し、より直感的で使いやすい操作性を実現
- マウスホイールでのズーム操作の改善
- 中ボタン＋ドラッグ パン（画面平行移動）。現状はA/Dキーのみだった。距離に応じ感度自動調整
- F キー	選択モデルにフォーカス（カメラを寄せてフレーミング）
- ヘルプ更新	"Commands" に新操作＋E/Q（上下）を追記
- [CUE4Parse-Conversion の作り直しのPRの実装](https://github.com/FabianFG/CUE4Parse/pull/358) ([4sval](https://github.com/4sval)のPRを実装)

すべての変更履歴については、[GitHub のリリース](https://github.com/Fortniteleakjp/FModel-JP-/releases) をご確認ください。
